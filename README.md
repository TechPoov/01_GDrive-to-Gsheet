# GDrive-to-Gsheet

GDrive-to-Gsheet is a TechPoov utility that scans a Google Drive folder
and exports file and folder details into a Google Sheet for easier analysis.

## Key Features
- Scans folders and subfolders in Google Drive.
- Exports results into a Google Sheet.
- Designed for non-developers and busy IT users.

## Getting Started
1. Go to the **Releases** section of this repository.
2. Download the latest `GDrive-to-Gsheet` zip file.
3. Unzip it and follow the included User Manual PDF.

For a shorter overview, see **QuickStart.md** in this repository.
